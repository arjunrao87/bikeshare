## Using the app [ *This app works only locally. As of now, you cannot deploy it to Heroku or AWS* ]

### Set up project

`git clone https://github.com/arjunrao87/bikeshare.git`
`cd bikeshare`
`npm install`
`cd client`
`npm install`
`cd ..`
